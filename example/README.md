# zklogin_dart example

A minimal, network-free walkthrough of the zkLogin building blocks — generating
an ephemeral key pair and nonce, and deriving a Sui zkLogin address from a JWT.

Run it with:

```sh
dart run example/zklogin_dart_example.dart
```

See [`zklogin_dart_example.dart`](zklogin_dart_example.dart) for the code, and
the `integration` test in the repository for the full prover + transaction flow.
